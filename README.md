# Spark Funds Investment Analysis
An end-to-end investment analysis using Crunchbase startup funding data to identify the optimal funding type, investment-friendly countries, and high-growth sectors.

## Overview

This project analyzes startup funding data from Crunchbase to help Spark Funds identify the best investment strategy based on historical funding trends.

The objective is to recommend:

- The most suitable funding type
- The best English-speaking countries for investment
- The most promising business sectors

The analysis was completed using Python, Pandas, and Matplotlib as part of the PG Diploma in Machine Learning and AI.

---

## Business Problem

Spark Funds wants to invest between **USD 5–15 million** per funding round and prefers investing in **English-speaking countries**.

Following its investment strategy of investing where other investors are investing, this project identifies the funding type, countries, and sectors receiving the highest investment activity.

---

## Dataset

The project uses three datasets:

- **companies.csv** – Company information
- **rounds2.csv** – Funding round details
- **mapping.csv** – Sector mapping
